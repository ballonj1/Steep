# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users
- `GET /api/users/:id`
- `POST /api/users`
- `PATCH /api/users/:id`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Events

- `GET /api/events`
- `POST /api/events`
- `DELETE /api/events/:id`

### Cities
- `POST /api/cities`
- `GET /api/cities/`
- `GET /api/cities/:id`

### Hosts

- `GET /api/hosts/`
- `POST /api/hosts/`
- `PATCH /api/hosts/:id`
- `DELETE /api/hosts/:id`
